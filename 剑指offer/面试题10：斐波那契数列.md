# 面试题10：斐波那契数列

## 题目描述

大家都知道斐波那契数列，现在要求输入一个整数n，请你输出斐波那契数列的第n项（从0开始，第0项为0）。

n<=39



## 解答

~~~java
public class Solution {
       int[] a = new int[40];

    public int Fibonacci(int n) {
        if (n == 1){
            return 1;
        }
        if (n == 0){
            return 0;
        }
        if (a[n] != 0) return a[n];
        a[n] = Fibonacci(n - 2) + Fibonacci(n - 1);
        return a[n];
    }
}
~~~
